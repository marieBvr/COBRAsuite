About COBRAsuite database

What is CAOBRAsuite ?

ANR Programme: Food & Feed: crop yields and nutrition security in the context of climate change (PLANT-KBBE) 2013
Project ID: ANR-13-KBBE-0006
Project coordinator: Veronique DECROOCQ (UMR 1332 Biologie et Pathologie du Fruit)

Data source

COBRAsuite gathers information from 3 data sources:

  • BIOGRID
  • INTACT
  • STRINGdb

Contributors

COBRAsuite plant virus interaction database is developed in coordination with other plant genomics and bioinformatics groups via the consortium. The Plant KBBE project is funded by the European Commission within its 7th Framework Programme, under the thematic area.

Website

The COBRAsuite website is powered by INRA.
The current page is using a bootstrap (Code licensed under MIT) template provided by Start bootstrap (license Apache 2.0). The web-application is based on Flask python framework. The network viewer is based on Cytoscape.js framework.
